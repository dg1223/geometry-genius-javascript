What is DOM? what is the purpose of DOM?

DOM stands for Document Object Model. It represents a tree stucture for HTML elements consisting of nodes and objects which in turn represents the entire page.DOM allows us to display web pages on a browser. It also lets us modify the content of the page.

How will you select elements from the DOM?

We can select elements from the DOM in 6 ways which are the following:

getElementById() - get HTML element by its id
getElementsByTagName() - get HTML element by its tag name
getElementsByClassName() - get HTML element by its class name
getElementsByName() - get HTML element by its name (attribute)
querySelector() - it returns the first HTML element of the selector
that is queried
querySelectorAll() - it returns all of the HTML elements of the selector that is queried

What is event bubble?

The event bubble is, in simple terms, propagation of JS events. After event listeners are dispatched, the event trickles down from one element to another through the bubble.

What is a callback function? What is it used for?

A callback function is a function that is passed as an argument to another function. It is used to perform operations that are otherwise repetitive in nature within other functions.
